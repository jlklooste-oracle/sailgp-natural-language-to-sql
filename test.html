Welcome to a demonstration of Natural Language to SQL (NL2SQL) and MySQL HeatWave Lakehouse. This demonstration uses Large Language Model (OpenAI) to translate your question into SQL, run the SQL on MySQL, and return an answer in natural language. The data you can work in this example is from a SailGP. The dataset contains a set of sensor values captured at every second of the race, including metrics like boat speed, wind angle, whether the boat is foiling, et cetera. You can ask questions like "What was the top speed of team USA?" or "How long did team New Zealand foil during the race?".
